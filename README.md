# YaoList 前端项目
基于 React 18 + TypeScript + Vite 的现代化前端应用。


## 打包指南



### nodejs

#### 安装
https://nodejs.org/zh-cn/download


#### 镜像配置
```shell
npm config set registry https://registry.npmmirror.com
```

### vite 安装

```shell
npm install -D vite
```


### 打包
```shell

npm run build
```

### 最终输出

/dist/
